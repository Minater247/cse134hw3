Description

If you were on the internet in 2020, you've probably heard of Friday Night Funkin'. It's a rhythm game, similar to Dance Dance Revolution or Osu Mania, but with the skill curve dialed way back. I always wanted a Scratch version of the game in High School, but all the existing ones were, for lack of words, Scratch games. I wanted a proper, nice-feeling version - and this was the result.

In the time since then, I've been expanding the engine to be even better. Now it takes 5 minutes to add your own song, 10 minutes to add a character, and it should be relatively simple to import charts from any engine across the entire existing FNF modding landscape!
